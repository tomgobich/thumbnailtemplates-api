# ThumbnailTemplates API

API used for ThumbnailTemplates Build Version 4.


## Usage

### start the server 
```js
npm start
```

### use nodemon to start the server 
```
npm run nstart
```

### use babel transform es6 => es5
```js
npm run build 
```

### run test
```js
npm run test
```

### run the build server
```js
npm run serve
```
